## blog 

A hugo site where I host my blog where I discuss various topics including technology, cloud native, gaming, and more. 

Uses the [re-terminal](https://github.com/mirus-ua/hugo-theme-re-terminal) theme.

### Useful Commands For Development:
```bash
# Init git submodules (must be done before running)
git submodule update --init --recursive

# Add new content
hugo new content ./posts/[POST-NAME].md

# Run locally
hugo server
```
